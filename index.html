Ubuntu · amd64 / arm64 · 정적 바이너리

hddmount-tui

1TB/2TB/4TB/8TB HDD를 안전하게 마운트하는 Go 기반 TUI + CLI 도구. SMART 체크, mdadm RAID1/RAID10, 비대화형 자동화까지 하나의 바이너리로.

설치하기 GitHub 릴리스 노트

주요 기능

안전한 마운트 마법사

루트 디스크 자동 제외, 파괴적 작업은 장치 경로 재입력으로 이중 확인, 시스템 핵심 경로 마운트 차단.

SMART 상태 체크

디스크 목록에서 SMART 헬스를 바로 확인하고, smartctl -a 상세 리포트를 그 자리에서 조회.

RAID1 / RAID10 구성

동일 용량 디스크를 골라 mdadm RAID 배열을 생성하고, mdadm.conf/initramfs까지 자동 갱신.

비대화형 CLI

프로비저닝 스크립트/cron에 바로 넣는 list/mount/unmount/format-disk 서브커맨드.

운영 로그 + 라벨

모든 마운트/언마운트/포맷 작업을 /var/log/hddmount.log에 기록, 마운트 포인트 기반 ext4 라벨 자동 설정.

fstab 안전망

등록 전 자동 백업, mount -a --fake 문법 검증, 실패 시 자동 롤백으로 재부팅 사고 방지.

설치

curl -fsSL https://LeeAn0121.github.io/hddmount-tui/pubkey.gpg \
  | sudo gpg --dearmor -o /usr/share/keyrings/hddmount-tui.gpg

echo "deb [signed-by=/usr/share/keyrings/hddmount-tui.gpg] \
https://LeeAn0121.github.io/hddmount-tui stable main" \
  | sudo tee /etc/apt/sources.list.d/hddmount-tui.list

sudo apt update
sudo apt install hddmount-tui

sudo hddmount

이후 새 버전은 sudo apt update && sudo apt upgrade 로 갱신됩니다.

TUI 조작

화면키
디스크 목록↑/↓ 이동 · enter 선택 · s SMART 상세 · R RAID 구성 · q 종료
파티션 선택↑/↓ 이동 · enter 마운트/포맷 · u 마운트 해제 · b 뒤로
확인/경고y/n 또는 방향키+enter, 파괴적 작업은 경로 재입력으로 이중 확인

자동화 CLI

sudo hddmount list --json
sudo hddmount smart --disk sdb --json
sudo hddmount mount --partition sdb1 --mountpoint /data/hdd1 --format --fstab
sudo hddmount unmount --mountpoint /data/hdd1
sudo hddmount format-disk --disk sdb --confirm sdb --mountpoint /data/hdd1 --fstab
